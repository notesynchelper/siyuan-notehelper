# 笔记同步助手

关注服务号使用。基础功能免费，进阶功能需付费订阅。

![preview](preview.png)

![关注公众号](https://siyuan.notebooksyncer.com/siyuanmp.png)

## 安装

1. 下载 `package.zip`
2. 解压到思源插件目录：`{工作空间}/data/plugins/`
3. 重启思源笔记并启用插件
4. 密钥：关注服务号发送"思源笔记"获取
